Packages

We publish also packages on the Python Package Index. Third party also publish modules for Tryton there.
You can directly install them using the pip command-line tool:


        $ pip install trytond
        $ pip install tryton
        $ pip install trytond_sale

Replace trytond_sale with the name of the module needed.

The web client can not be installed with pip, you must follow this installation procedure.

Distributions

Tryton is also packaged and distributed by third-parties volunteers. Often there is a delay with the release of new versions but the installation is more integrated with the operating system.
